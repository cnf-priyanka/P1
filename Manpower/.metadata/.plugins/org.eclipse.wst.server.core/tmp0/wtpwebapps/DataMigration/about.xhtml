About DataMigrator

  • There are numerous benefits of using PostgreSQL instead of DB2, with it open-source and the most cross-platform of them all.

  • DataMigrator is a tool that allows you to easily and seamlessly migrate and transfer data between the two databases.

  • You can create the connection to the databases, providing the details (name and URL of the database, username and password).

  • Also, on the clean page, you can clean out the destination database i.e., the PostgreSQL database to get rid of any anomalies before you begin the migration.

  • Next, the Migration page allows you to select the tables which you want to migrate from DB2 database, into the PostgreSQL database.

  • The stats pages are there to view the rows count of both the databases and also have the option to view a direct comparison between them.
